## 🛠 Projects

**[Multi-System Customer Data Integration & Entity Resolution](https://github.com/devika1402/Multi-System-Customer-Data-Integration-Entity-Resolution)**
Integrated Stripe, Salesforce, and NetSuite datasets (~2M rows). Resolved 98% of foreign key mismatches using rule-based entity resolution and cross-system reconciliation logic.
`Python` `SQL` `Entity Resolution` `Data Quality`

**[Lead Conversion Analytics](https://github.com/devika1402/lead-conversion-analytics)**
End-to-end data product: containerised Postgres database, API-driven synthetic data generation, SQL KPI queries, and a dashboard prototype for conversion funnel monitoring.
`PostgreSQL` `Docker` `SQL` `Dashboard Design`

**[Fashion Sales Analyzer](https://github.com/devika1402/fashion-analyzer)**
PySpark pipeline analyzing fashion sales across category, style, season, and region. Exported curated insights to CSV/JSON for stakeholder reporting.
`PySpark` `Python` `Data Visualization`

**[Graph Anonymization Study](https://github.com/devika1402/privacy-by-design)**
Evaluated 4 anonymization methods across 5 real-world graphs. Measured structural preservation (modularity shift <3%) and re-identification risk (<1%).
`Python` `Graph Theory` `Privacy`
